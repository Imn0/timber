a C logging library

format specification
    * %m - log message
    * %n - loggers name
    * %l - log level (long)
    * %L - log level (short)
    * %s - basename source file (like main.c)
    * %@ - source file (like /src/main.c)
    * %# - source line
    * %f - function
    * %t - logger tags
